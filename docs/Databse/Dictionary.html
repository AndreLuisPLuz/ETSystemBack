Tabela person
Descrição Representa uma pessoa. Especializa-se em student, instructor e administrator.
Observações
Campos
Nome Descrição Tipo de Dado Tamanho Restrições de Domínio (PK, FK, Not Null, Check, Default, Identity)
id_person Identificador único de person uniqueidentifier PK, NOT NULL
name Nome da pessoa VarChar 255 NOT NULL
email E-mail para contato VarChar 255 NOT NULL
date_of_birth Data de nascimento Date NOT NULL
contact Número de telefone, sem parênteses ou traços. VarChar 20 NOT NULL
access_type O tipo de acesso da pessoa, definido nos seguintes níveis: [1 - Senai | 2 - Bosch | 3 - Master]. Para estudantes, somante o nível 1 está disponível. Para instrutores, níveis 1 e 2. VarChar 20 NOT NULL
is_active Indica se a pessoa está ativa ou não. Bit NOT NULL
